2023：有些许焦虑的一年

发表于 2023-12-27 分类于 叙年 阅读次数： 本文字数： 606 阅读时长 ≈ 1 分钟

年初时定了一些目标：

  1. 好好对女朋友
  2. 新冠结束，有机会多回家晃悠晃悠
  3. 书3本（偏向JAVA以前的书重看，重新学习总结）
  4. 看开源项目源码学习（部分可以动手写代码验证），加强JAVA代码能力
  5. 保持技术阅读（以ATA为主，尝试阅读论文（基本没有阅读过多少论文，需要多读训练））

总结：这是有些许焦虑的一年。

百年未有之大变局开始了
俄乌冲突局势明朗，巴以冲突……
QQ 音乐给的关键词是世界
搬着小板凳等着这世界改变

这一年
房价开始下降了，等等党好像不输
拼多多市值超过阿里了
我开始用文心一言替代普通搜索，这算是大模型对搜索的冲击了
云行业环境变差了，有些许焦虑

今年我体会到在大型项目架构上经验的缺乏
在代码构建上方式单一，无法游刃有余
终于开始看一些开源项目代码
深切理解了组织架构对工程架构、效能的影响
开始看软件工程的书，但看完书量为 0

“大感冒” 完全结束了
我终究 🐑 过了
这一年也走出去过，去过厦门、杭州、西昌
也终于是带爸妈去看了熊猫，超羡慕熊猫没有 KPI 的躺平生活呀

终究发现，最尴尬的是上不去又躺不平的普通人的生活
幸运的是，人总比想象的坚强，哪怕再渺茫的希望，也会去求索
接受现实后依旧微笑面对生活，才算是乐观派吧
我也能接受现实，也算是乐观派吧

来年目标：

  1. 依旧好好对女朋友
  2. 书 3 本。补足软件工程宏观或启发式缺漏，降低认知不足导致的偏见
  3. 阅读开源项目至少 4 个。学习多种架构，增强更多的架构能力；以及解决构建代码方式单一，无法游刃有余问题
  4. 有空多出去走走，不设目标
